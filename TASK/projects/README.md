# 子项目专项文档索引

> 文档职责：本目录收口“跨子项目阅读才需要的专项材料”。  
> 子项目根目录只保留轻量入口文档；深入分析、迁移记录、专项矩阵、协议文档、设计补充统一放到这里做索引。

## 子项目入口

- `aigateway-console/README.md`
  - Console GoFrame 重写、Java parity、Provider / AI Route / MCP / 协议专项材料入口
- `aigateway-portal/README.md`
  - Portal 当前项目说明与根目录索引入口
- `higress/README.md`
  - Higress 当前项目说明与根目录索引入口
- `plugin-server/README.md`
  - Plugin Server 当前项目说明与根目录索引入口

## 使用原则

- 子项目内长期沉淀的专项材料优先迁到 `TASK/projects/<subproject>/`
- 子项目根目录的 `TASK.md` 只做轻量导览，不再承载长篇专项记录
- 若某专项同时影响多个子项目，应优先放根目录并在相关子项目索引中互相引用
