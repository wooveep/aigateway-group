# AIGateway 1.0.0 用户手册

## 1. 适用范围

本手册面向 `1.0.0` 正式交付版本，覆盖：

- Portal 访问入口与主要用户功能
- Console 访问入口与主要控制面功能
- 截图资产目录：`output/manual/1.0.0/`

## 2. 访问入口

- Console
  - 默认本地入口：`http://127.0.0.1:8080`
- Portal
  - 默认本地入口：`http://127.0.0.1:8081`
